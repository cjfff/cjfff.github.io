> Markdown 是什么：

* Markdown 是一种非常简单的、轻量级的标记语言。用户可以使用简单的标记符号**以最小的输入代价去生成极富表现力的文档**。Markdown
  的语法简洁明了、学习容易，功能比纯文本强得多，因此有很多人用它写博客、写文档。

---

> Markdown 具有很多优点：

* 写作中添加简单符号即可完成排版，所见即所得，让你专注于文字而不是排版。
* 格式转换方便，Markdown 的文本你可以轻松转换为 HTML、PDF 等文件。
* 自带格式化/排版信息，且规则简单友好，可保存为纯文本。
* 对比 HTML：它更专注于内容，标签对内容的侵入性低。
* 写作时舒服，优雅感，你会感觉它就是为博客而生。

### Markdown 使用语法如下:

#### 1.[标题]一级标题-六级标题

语法: # 标题名称

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

效果如下:

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

注:# 和一级标题之间要保留一个空格(其它部分语法也同样需要加空格),因为这才是最标准的 Markdown 写法。

---

#### 2.[列表]无序列表-有序列表

无序列表语法: - 名称

```
- 我是无序列表
- 我是无序列表
- 我是无序列表
```

效果如下:

* 我是无序列表
* 我是无序列表
* 我是无序列表

---

有序列表语法: 1. 名称

```
1. 我是有序列表
2. 我是有序列表
3. 我是有序列表
```

效果如下:

1.  我是有序列表
2.  我是有序列表
3.  我是有序列表

---

#### 3.链接

语法: `[名称](链接地址)`
如:
`[百度一下,你就知道](http://www.baidu.com)`
效果如下:
[百度一下,你就知道](https://link.jianshu.com?t=http://www.baidu.com)

---

#### 5.引用文字(注意是文字!不是代码,因为有专门的引用代码语法,还在后面)

注:在我们使用 Markdown 写作的时候经常需要引用他人的文字,这个时候引用语法就很有必要了,在 Markdown 中,你只需要在你希望引用的文字前面加上 > 就好了,例如:

`> 有了引用文字语法,读者在看我的文章时可以明显看的清晰明了~`

效果如下:

> 有了引用文字语法,读者在看我的文章时可以明显看的清晰明了~

---

#### 6.粗体-斜体

注:用两个 \* 包含一段文本(也就是开始和结尾)就是粗体的语法,如:

`**劳资翻身做码农**`

**劳资翻身做码农**

注:用一个 \* 包含一段文本(也就是开始和结尾)就是斜体的语法,如:

`*劳资翻身做地主*`

_劳资翻身做地主_

---

#### 7.代码引用(码农最爱啦~)

注:假设你引用的代码只有一段,不分行,那么可以用 ` 将此段代码前后(首字前末字尾)包起来.
如:

```
` hello world `
```

效果如下:
`hello world`

<br>
注:假设你引用的代码很多段,很多行,那么可以用```将此段代码前后(首字前末字尾)包起来.
效果如下:

```
public class Hello{
public static void main(String args[]){
int age = 16 ;
if(age < 18){
System.out.println("少儿不宜") ;
}
}
}
```

#### 8.做表格

注:比较以上语法来说稍有点绕,但是不难理解.如:

```js
| 吃 | 喝 | 玩 |
| --- | --- | ---- |
| 炸鸡 | 啤酒 | 欧洲游 |
| 烧烤 | 香槟 | 非洲游 |
| 火锅 | 雪碧 | 家里游 |
```

效果如下:

| 吃   | 喝   | 玩     |
| ---- | ---- | ------ |
| 炸鸡 | 啤酒 | 欧洲游 |
| 烧烤 | 香槟 | 非洲游 |
| 火锅 | 雪碧 | 家里游 |

注:表格标题会默认居中,但内容不会居中.如果想要让内容居中的话,请参考如下代码:

```
| 吃 | 喝 | 玩 |
| :---: | :---: | :---: |
| 炸鸡 | 啤酒 | 欧洲游 |
| 烧烤 | 香槟 | 非洲游 |
| 火锅 | 雪碧 | 家里游 |
```

效果如下:

| 吃                          | 喝     | 玩                                       |
| --------------------------- | ------ | ---------------------------------------- |
| 炸鸡                        | 啤酒   | 欧洲游                                   |
| 烧烤                        | 香槟   | 非洲游                                   |
| 火锅                        | 雪碧   | 家里游                                   |
| 注:只需要在第二行里面每个的 | \-\-\- | 里面加上一对儿::(英文冒号)即可内容居中.~ |
